# fastMCP_tutorial

這個 repo 可以直接用 GitHub Pages 發布成靜態網站。

這不是個人首頁設定，不需要把 repo 命名成 `你的帳號.github.io`。這是一般專案網站，發布後網址會像：

```text
https://你的帳號.github.io/這個repo名稱/
```

## GitHub Pages 設定

1. 把這個 repo push 到 GitHub。
2. 進入 GitHub repo 的 `Settings`。
3. 左側選 `Pages`。
4. `Source` 選 `Deploy from a branch`。
5. `Branch` 選 `main`，資料夾選 `/ (root)`。
6. 按 `Save`。

發布後首頁會讀取 `index.html`，並自動導向：

```text
mcp_fastmcp_intro_course.html
```
